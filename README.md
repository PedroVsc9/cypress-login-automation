# cypress-login-automation

Projeto de automação de testes E2E para o fluxo de login do site [Automation Practice](https://automationpratice.com.br/login), desenvolvido com Cypress e JavaScript.

---

## Cenários cobertos

- Login com credenciais válidas
- Login com senha incorreta
- Login com campos vazios
- Validação de mensagens de erro

---

## Tecnologias

- [Cypress](https://www.cypress.io/)
- JavaScript

---

## Como rodar

```bash
# Instalar dependências
npm install

# Rodar os testes no navegador
npx cypress open

# Rodar os testes em modo headless
npx cypress run
```
